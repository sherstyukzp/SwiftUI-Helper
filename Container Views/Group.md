Tag: #Group #Text 

---
## Описание:
Группа используется для объединения нескольких представлений, и свойства, установленные для группы, будут применяться к каждому дочернему представлению.

---
## Код:

``` swift
Group {
    Text("Hello World !")
    Text("Hello World !")
    }

```
